## 在线视频自动分段项目
### Step1:
    从视频文件中抽取音频
    - a 视频文件的格式
    - b 音频文件（统一到.wav）
    - c 录音生成.wav (实验）
### Step2:
    把音频文件识别成文本
    - a 语音识别Kalid
    - b 文本纠错
### Step3:
    提取出文本的摘要
    - a Cut into Num Parts
    - b TFIDF/PageRank/TextRank 
    - c Keywords Produce
### Step4:
    Map文本摘要到视频的各段，完成项目
    Record Time stamp for each Parts
